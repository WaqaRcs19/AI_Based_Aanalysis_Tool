# AI_Based_Aanalysis_Tool
Ai powered streamlit data analysis tool that can do basic data analysis and we can also create some basic visualization  from it

🏃 How to Run
Clone or download this repository.

Navigate to the project directory in your terminal.

Run the application:

Upload a CSV file through the sidebar to begin your analysis.

💡 Future Enhancements
OpenAI API Integration: Fully connect the "Ask Questions" feature to an LLM to receive automated textual answers about data trends.

Advanced Filtering: Add sidebar filters to slice data before visualization.
